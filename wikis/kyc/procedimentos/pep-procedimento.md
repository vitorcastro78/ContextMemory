# Procedimentos KYC

## Cliente PEP — Procedimento operacional

Quando um cliente é classificado como PEP, o analista deve seguir o fluxo abaixo.

### Passos iniciais

1. Confirmar a classificação PEP na base de dados interna
2. Recolher documentação de origem de fundos
3. Submeter caso para aprovação do Compliance Officer

### Documentação mínima

- Identificação válida do titular e UBOs
- Comprovativo de morada (últimos 3 meses)
- Declaração de origem de fundos assinada

### Prazos

O processo deve ser concluído em até 10 dias úteis após identificação PEP.
